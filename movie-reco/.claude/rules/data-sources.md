# Règle : sources de données

Ce projet est destiné à un usage commercial. Toute donnée intégrée au produit doit
être libre pour un usage commercial ET redistribuable.

AUTORISÉ dans le produit :
- Wikidata (CC0) pour les métadonnées.
- Wikipedia / DBpedia (CC BY-SA) pour le texte, avec attribution et partage à l'identique.

INTERDIT dans le produit (restrictions commerciales ou autorisation requise) :
- MovieLens / GroupLens
- Datasets IMDb
- API Trakt
- TMDB

Ces sources interdites peuvent éventuellement servir à des tests ou comparaisons
**en local et en privé**, jamais dans le livrable distribué. En cas de doute sur une
nouvelle source, ne pas l'ajouter sans valider sa licence d'abord.
